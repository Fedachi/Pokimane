Rules

Server Rules:
DO NOT TAG POKIMANE

It results in spam notifications for her and it can get you kicked from this server.

Hate speech will not be tolerated
This includes homophobia, xenophobia, discrimination of gender, racism, etc. Be kind to each other.

Zero tolerance for NSFW / NSFL content / discussions.
This includes usernames and avatars

No Spamming
This includes emotes, text, images, and commands (commands go in #bot-commands).

No Advertisements or promotions of any kind in this server or in DMs
This includes Discord servers, social media, content channels etc

Encouraging others to break any of the rules is not allowed
English is the main language in this server - please keep all conversations in English
French can be spoken in the #français

Any Impersonation will result in a kick.
​ This includes the impersonation of staff and bots.
​ Impersonation of Pokimane may result in a ban.

Use Common Sense.

If you have any issues, please let a moderator or admin know.
Special ranks are earned, not given freely. Don't ask for any specific rank unless your sub role isn't showing up - if persists after an extended amount of time, make sure to let a mod/admin know.
User Names will be automatically changed if they're unmentionable or against the rules.
The everyone/here pings are disabled. Just don't.
Suggestions are always welcome.

Moderators may moderate at their own discretion.
Nothing is perfect, that includes our rules list. Any loopholes will not be tolerated. Use your common sense and do not complain when the staff team is trying to keep the server a calm and safe place.

Since this server is a Discord Partnered server. You must adhere to Discord ToS and Community Guidelines.
Failure to do so will result in reporting to Discord for review.

This Discord is a server for a partnered Twitch channel, and thus adheres to the Twitch Terms of Service, found Twitch ToS Twitch Community Guidelines.
Failure to adhere will result in a ban from the server and the stream chat.
